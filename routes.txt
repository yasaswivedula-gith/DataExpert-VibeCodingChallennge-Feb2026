# TaskFlow AI - Next.js Routes

## Public Routes
/                       # Landing page (Home)
/auth/login            # Login page
/auth/signup           # Signup page
/auth/error           # Error page

## Protected Routes (require authentication)
/dashboard             # Main dashboard (Tasks tab)
/dashboard/tasks       # Tasks management
/dashboard/workflows   # Workflows management  
/dashboard/ai-suggestions  # AI Assistant suggestions

## Server Actions (implemented)
/api/auth              # Handled via Supabase Auth
/api/tasks            # Task CRUD via Server Actions (src/lib/actions/tasks.ts)
/api/workflows         # Workflow management via Server Actions

## Route Protection
- Public: /, /auth/*
- Protected: /dashboard/* (redirect to /auth/login if not authenticated)
- Middleware: middleware.ts with matcher for all routes

## Tech Stack
- Next.js 16.1.6 (App Router)
- React 19.2.3
- Tailwind CSS 4
- TypeScript 5
- Vercel (Deployment)
